# Streamlit AI Chat App

A feature-rich AI chat application built with Streamlit.

## Features

- **Multi-Provider Support**: Integrates with Google Gemini, OpenAI, and Anthropic.
- **Chat Export (Experimental)**: Export your chat history to JSON format.
- **Internet Search**: Supplement AI responses with real-time search results.
- **Multimodal capabilities**: Support for image inputs.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the app:
   ```bash
   streamlit run app.py
   ```

## Experimental Features

- **Chat Export**: Currently in beta. Allows users to download their conversation history.
